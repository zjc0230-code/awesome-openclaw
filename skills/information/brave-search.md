
### 2️⃣ 配置 OpenClaw

**方法 A：命令行配置**
```bash
# 进入配置模式
openclaw configure --section web

# 输入 API Key（粘贴后不会显示）
Enter Brave Search API Key: [粘贴你的 API Key]

# 验证配置
openclaw ask "测试搜索：今天的科技新闻"
```

**方法 B：修改配置文件**
```bash
# 编辑配置文件
nano ~/.openclaw/openclaw.json

# 添加以下内容：
{
  "tools": {
    "web": {
      "search": {
        "provider": "brave",
        "apiKey": "YOUR_API_KEY_HERE",
        "region": "US",  // 地区代码（US/CN/GB 等）
        "language": "zh-Hans"  // 语言代码
      }
    }
  }
}

# 保存并重启 OpenClaw
openclaw restart
```

## 📚 使用技巧

### ✅ 最佳实践

1. **明确搜索意图**
   - ✅ "搜索 2026 年 AI Agent 最新论文"
   - ❌ "AI 相关内容"（太宽泛）

2. **使用时间范围**
   - ✅ "最近 24 小时关于 OpenAI 的新闻"
   - ✅ "2025 年发表的 Transformer 论文"

3. **结合多个关键词**
   - ✅ "React + TypeScript + 性能优化"
   - ❌ 单独搜索 "React"（结果太多）

4. **指定地区/语言**
   - ✅ "搜索日本市场对某产品的评价（日语）"
   - ✅ "搜索欧洲 GDPR 相关新闻（英语/德语）"

5. **利用 AI 分析**
   - ✅ "搜索并总结前 10 条结果"
   - ✅ "搜索并生成对比表格"

### ⚠️ 常见陷阱

1. **API 配额耗尽**
   - 问题：免费版 2,000 次/月用完
   - 解决：升级付费版（$5/月 20,000 次）或使用缓存

2. **地区限制**
   - 问题：某些地区无法访问 Brave API
   - 解决：使用代理服务器或 VPN

3. **搜索结果过多**
   - 问题：返回几千条结果无法处理
   - 解决：使用更精确的关键词或时间范围

4. **语言识别错误**
   - 问题：搜索中文内容返回英文结果
   - 解决：明确指定语言参数

5. **实时性不足**
   - 问题：某些事件搜索不到（如 5 分钟前发生）
   - 解决：等待 10-30 分钟后再搜索（索引延迟）
