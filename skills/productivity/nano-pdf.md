## 🎯 核心功能

### 1. PDF 合并
```
用户: 将 'contracts' 文件夹中的所有 PDF 合并为一个文件，命名为 'Q1_Reports.pdf'
```

### 2. PDF 拆分
```
用户: 从 'research.pdf' 中提取第 5-10 页，保存为 'excerpt.pdf'
```

### 3. 文本提取
```
用户: 提取 'report.pdf' 中的所有文字内容，保存为文本文件
```

### 4. 格式转换
```
用户: 将这个 PDF 转换为 Word 文档格式
```

### 5. 内容编辑
```
用户: 在 'document.pdf' 的第 3 页添加水印
```

## 🔧 故障排查

### 问题 1: 找不到文件

**错误信息**: `Error: File not found: document.pdf`

**解决方案**:
1. 检查文件路径是否正确
2. 确认文件名拼写
3. 尝试使用完整路径

---

### 问题 2: 权限错误

**错误信息**: `Error: Permission denied`

**解决方案**:
1. 检查文件是否被其他程序占用
2. 确认文件可写权限
3. 关闭 PDF 阅读器后重试

---

### 问题 3: 处理失败

**错误信息**: `Error: Processing failed`

**解决方案**:
1. 确认 PDF 文件未损坏
2. 检查文件大小（建议 < 100MB）
3. 尝试分批处理
